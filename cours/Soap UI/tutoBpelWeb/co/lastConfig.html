Dernières actions de configuration pour ODE

Ajoutez un nouveau serveur à votre Workspace. Sélectionnez un serveur Apache ODE puis cliquez sur "Next >".

Ajoutez votre BPEL Process File au déploiement du serveur et cliquez sur "Finish".

Dans le panneau "Servers", double-cliquez sur le serveur ODE pour faire apparaître son "Overview" dans le panneau central. Puis, cliquez sur "Open launch configuration :"

Sélectionnez "User Entries" puis cliquez sur "Add External JARs...". Un explorateur de fichier s'ouvre...

Dans l'explorateur de fichier, rendez-vous dans le répertoire "bin" de votre serveur Tomcat ODE et sélectionnez les jars "commons-daemon.jar" et "tomcat-juli.jar" avant de cliquer sur "Ouvrir".

Une fois les deux JARs ajoutés au Classpath de votre ODE, cliquez sur "OK".

Dernière étape de configuration. Ouvrez le WSDL File de votre orchestrateur en vue Source et cherchez la balise "<soap:address>". Modifier l'URI dans l'attribut "location" en ajoutant : "ode/processes/" entre le port du serveur et le nom de votre BPEL Process File. C'est à cette adresse que votre serveur Tomcat ODE rendra disponible le processus d'orchestration à l'application cliente.
