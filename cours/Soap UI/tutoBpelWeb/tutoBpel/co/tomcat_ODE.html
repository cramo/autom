Installation du moteur d'orchestration ODE

Apache ODE est un moteur d'exécution de processus BPEL . C'est une implémentation de la norme WS-BPEL. Nous allons installer ce moteur d'orchestration sur un serveur apache Tomcat.

Téléchargement de l'archive web ODE

Rendez-vous sur http://ode.apache.org/ . Dans l'onglet "Getting ODE", téléchargez l'archive "apache-ode-war-1.3.6.zip".

Extraction de l'archive web ODE

Dézippez l'archive téléchargée. Dans le répertoire racine du dossier obtenu, copier le fichier "ode.war". Nous allons déployer cette archive war sur un serveur Tomcat pour y installer le moteur d'orchestration ODE.

Préparation du déploiement de l'archive web ODE sur Tomcat

Collez maintenant le fichier "ode.war" dans le répertoire "webapps" de votre serveur Tomcat.

Déploiement de l'archive web ODE sur Tomcat

Dans le répertoire "bin" de votre serveur Tomcat, double-cliquez sur l’exécutable "startup.bat" afin de démarrer votre serveur et de déployer "ode.war".

Votre serveur a démarré lorsque la ligne se terminant par "Server startup in xxxxx ms" s'affiche. La ligne ici encadré en rouge indique qu' "ode.war" a bien été déployé sur le serveur. Vous pouvez désormais éteindre votre serveur avec ctrl+c .

Vérification de l'installation du moteur d'orchestration ODE

Pour vérifier que le déploiement ait bien fonctionné, retournez dans le répertoire "webapps" de votre serveur Tomcat. Vous devriez y trouver un nouveau répertoire nommé "ode". Il contient tout le nécessaire pour que votre serveur Tomcat puisse désormais exécuter du code BPEL. Votre moteur d'orchestration est prêt à l'emploi !
